# 1. Introducción

Todas las páginas web utilizan internamente los lenguajes de marcas. También algunos de los programas que podemos tener instalados en nuestros dispositivos.

Existen distintas formas de representar estar marcas o etiquetas, las más utilizadas son los símbolos "<" y ">".

XML (eXtensible Markup Language, en español lenguaje de marcas extensible).

XML es un metalenguaje de etiquetas desarrollado por W3C (World Wide Web Consortium – consorcio internacional que crea recomendaciones para la WWW).

¿Qué es XML?

XML no es un lenguaje, es un metalenguaje, puesto que, permite crear lenguajes.
